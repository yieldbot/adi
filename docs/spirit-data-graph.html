1    Introduction

spirit.data.graph provides methods for outputting data from nested structures

1.1    Installation

Add to project.clj dependencies:

[im.chit/spirit.data.graph "0.7.1"]

All functions are in the spirit.data.graph namespace.

(require '[spirit.data.graph :as graph])
