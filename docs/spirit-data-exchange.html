1    Introduction

spirit.data.exchange provides an interface that makes working with exchanges and exchange-like interfaces easy. Currently, the interface supports cassandra, dynamodb, redis and an atom-backed store used for mocking.

1.1    Installation

Add to project.clj dependencies:

[im.chit/spirit.data.exchange "0.7.1"]

For interfacing with cassandra, dynamodb and redis, please also include the relevant dependencies: 

[im.chit/spirit.rabbitmq  "0.7.1"]

All functions are in the spirit.data.exchange namespace.

(require '[spirit.data.exchange :as exchange]
         '[spirit.rabbitmq :as rabbitmq])

  ([mq source dest opts]

publish ^

publishes a message to an exchange

v 0.5
(defn publish
  ([mq exchange message]
   (publish mq exchange message {}))
  ([mq exchange message opts]
   (exchange/-publish mq exchange message opts)))
link
(def p (promise))
  
(-> (exchange {:type :mock
               :routing routes
               :consumers {"q1" {:hello {:function #(deliver p %)}}}})
    (publish "ex1" "hello there"))
  
@p => "hello there"
